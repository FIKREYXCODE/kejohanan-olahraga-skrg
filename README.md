# Kejohanan Olahraga Tahunan SK Ranggu

Laman rasmi Sistem Kejohanan Olahraga Tahunan Sekolah Kebangsaan Ranggu, Peti Surat 842, 91008 Tawau, Sabah.

## Cara admin mengubah data

1. Buka fail `data.json` dalam repositori ini.
2. Tekan ikon pensel **Edit this file**.
3. Pilih tahun bermula 2026.
4. Ubah `housePoints`, `results`, atau `schedule`.
5. Tekan **Commit changes**.

Contoh rekod murid:

```json
{
  "athlete": "Nur Aisyah Binti Ali",
  "event": "100m Tahun 6 Perempuan",
  "house": "Merah",
  "category": "Balapan",
  "place": 1,
  "mark": "13.25s"
}
```

Hanya pengguna GitHub yang mempunyai kebenaran menulis pada repositori boleh menyimpan perubahan. Orang ramai masih boleh melihat laman melalui GitHub Pages.
